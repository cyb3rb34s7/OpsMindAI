## Known gaps (Python dynamic dispatch)

Static analysis cannot resolve:
- `AGENT_REGISTRY.get(agent_name)` → which agent class
- `agent.run(context, payload)` → `BaseAgent.run()` → `agent.execute()`
- `runner = CognitiveRunner(...)` → `runner.run(...)` — type lost on local var
- `self.log_tool.execute(...)` — attribute type unknown

These call chains exist in the code but are invisible to tree-sitter.
See `docs/code-analysis-gaps.md` for details.
